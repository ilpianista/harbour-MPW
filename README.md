## Build

To install `libmpw`, performs the following steps:

```
sfdk tools exec SailfishOS-4.1.0.24-i486
ssu ar ilpianista http://repo.merproject.org/obs/home:/ilpianista/sailfishos_latest_latest_i486/
zypper ref -f
```
